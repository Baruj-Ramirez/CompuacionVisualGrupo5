## 📖 Acerca de este Repositorio

Este repositorio contiene las entregas de talleres y proyectos del curso de Computación Visual, implementando conceptos fundamentales y avanzados de gráficos por computador. Cada taller explora diferentes aspectos del pipeline gráfico utilizando múltiples tecnologías.

## ✅ Buenas Prácticas

### Estructura de Entregas

Cada taller debe incluir:
- 📄 `README.md` - Documentación de la actividad y aportes individuales
- 💻 Código fuente organizado por tecnología (`python/`, `unity/`, `threejs/`)
- 🖼️ `media/` - Capturas de pantalla, videos o GIFs demostrativos

### Commits

Todos los commits se realizan en inglés siguiendo buenas prácticas:

✅ **Correctos:**
```
Add rasterization algorithms implementation
Fix depth buffer rendering issue
Update README with project documentation
```

❌ **Incorrectos:**
```
subí lo mío
último intento
cambios
```

### .gitignore

El repositorio está configurado para ignorar archivos innecesarios:
- `node_modules/` (Node.js/Three.js)
- `Library/`, `Temp/` (Unity)
- `.ipynb_checkpoints/`, `__pycache__/` (Python)

## 🚀 Inicio Rápido

### Proyectos Python/Jupyter

```bash
cd semana_XX_Y_nombre_taller/python/
jupyter notebook
```

### Proyectos Three.js

```bash
cd semana_XX_Y_nombre_taller/threejs/proyecto/
npm install
npm run dev
```

### Proyectos Unity

Abrir el proyecto desde Unity Hub y explorar las escenas en la carpeta `Assets/Scenes/`
